# V2Board

[V2Board](https://github.com/v2board/v2board) 是近年新开发的又一个机场面板之一，因界面优雅、配置明了而深受广大机场主喜爱。

## 功能特性

- 支持 Shadowsocks、Vmess、Trojan 协议
- Laravel 框架开发
- 稳定、简单、快速
- 单一版本


## 安装

### 环境依赖

- PHP >= 7.3
- Nginx >= 1.17
- MySQL = 5.x
- Redis

基础环境安装参见 [Linux 基础](../linux.md)。
